# Leetcode (912. Sort an array)

### Description  

Given an array of integers `nums`, sort the array in ascending order.

### Note  

1. 1 <= A.length <= 10000
2. -50000 <= A[i] <= 50000

### Code  

~~~Javascript
let sortArray = (nums) => {
    nums.sort((e1, e2) => {
        if( isNaN(e1 - e2) ) 
            throw Error("Argument type is wrong.");
        return e1 - e2;
    });
    return nums;
};
~~~
